# Ethics & Compliance Documents

This folder contains IRB and PDPA compliance documents for the study.

## Contents (to be added by the researcher)

- `irb_approval.pdf` — IRB approval letter from KMUTNB Human Research Ethics Committee
- `informed_consent_th.pdf` — Participant consent form (Thai)
- `informed_consent_en.pdf` — Participant consent form (English)
- `pdpa_compliance.pdf` — PDPA (B.E. 2562) compliance statement
- `data_processing_agreement.pdf` — Data processing agreement for AI API usage

## Note

These documents are institutional records and must be obtained from
the KMUTNB research ethics office. They are not generated by the
software system. Please contact the corresponding author for access
during the peer-review process.
